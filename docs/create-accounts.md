You’ll want a few things in order before getting your Corporate Credentials and Company Laptop:

---

## GitHub Account (required)

If you don’t have one already, sign up for a **free** GitHub account.

> 💡 **Username / Profile Picture**  
> As your GitHub account is part of your professional profile, choose your GitHub username carefully (ideally it should reflect your real name).  
> If it doesn’t, we **strongly suggest** [changing it](https://docs.github.com/en/account-and-profile/setting-up-and-managing-your-personal-account-on-github/managing-user-account-settings/changing-your-github-username).  
>  
> Consider adding a professional profile picture.  
>  
> *(Using your personal GitHub account is OK. You’ll be contributing to open-source projects, and this becomes part of your professional profile — which should help when looking for a job after graduation.)*

- [Advice in choosing your GitHub Handle](https://www.notion.so/GitHub-Handle-LinkedIn-URL-2f7c973c93d1807ebd0cc01496335109?pvs=21)
- [Advice in choosing your profile image for GitHub](https://www.notion.so/GitHub-LinkedIn-Profile-Image-2f7c973c93d18021a6addaf8b4edeba0?pvs=21)
- [GitHub Account Sign Up](https://github.com/signup)

## GitHub Global Campus (required for co-ops)

GitHub provides students with several free services, many of which you will use during your co-op at Intealth.  
*(Requires a valid student ID — takes a couple of days to process)*

- [Apply to GitHub Global Campus](https://docs.github.com/en/education/explore-the-benefits-of-teaching-and-learning-with-github-education/github-global-campus-for-students/apply-to-github-global-campus-as-a-student#applying-for-a-github-student-developer-pack)

---

## 1Password (required for co-ops)

Security is essential. The GitHub Student Developer Pack provides **1 year of free access to 1Password**.

- [Get 1Password for free](https://1password.com/developers/students)  
  **(Free to CO-OPs only)**

---

## Secure your GitHub Account (required)

You must enable Multi-Factor Authentication (MFA) to protect your account.

You can use:
- 1Password  
- GitHub Mobile  

- [Add Authenticator App in GitHub Security Settings](https://github.com/settings/security)

---

## Apple (required)

You will most likely be issued a MacBook Pro and will need an Apple ID.

*(You can use your personal Apple ID if you already have one)*

- [Create an Apple ID](https://support.apple.com/en-us/HT204316)

---

## SonarCloud (required)

Signing up for SonarCloud helps ensure your code meets quality and security standards.

- [Sign up for SonarCloud using GitHub](https://www.sonarsource.com/products/sonarcloud/signup/)
